Manifest
--------

**Android 7.1.x** (latest is 7.1.1_r4)

1. Initialize the manifest repo		
`repo init -u https://github.com/krexus/manifest.git -b n+`

2. Sync		
`repo sync`

3. Lunch		
`lunch`

4. Build!		
`mka otapackage`
